Learn

The Learn Section provides predefined learning modules for children. Under the learn section, we find modules for english alphabets, numbers, colours, animals and birds. We also provide Read-Aloud feature to make verbal and visual learning fun.

Rhyme Room

The Rhyme Room section is for nursery rhymes and limericks for children. It will make learning fun and interesting. We also look forward to introducing traditional folk songs and music for children.

Learn Shapes

The Learn Shapes section, helps identify children different kinds of shapes. With development and advancement, we plan to add computer vision to the section, so that children can identify shapes using ML.

Detect Objects

The Detect Object section is a machine learning based section where we have integrated computer vision to enable object detection for children. Children can put forward any object and the model will help identify the objects by using camera.
